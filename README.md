# kickstart.nvim

## Introduction
Welcome to the customized Neovim configuration! This project is a fork of the kickstart.nvim configuration, customized for personal use.

## Installation
### Install kickstart.nvim
```sh
git clone git@github.com:mugi-02050x/kickstart.nvim.git "${XDG_CONFIG_HOME:-$HOME/.config}"/kickstart.nvim:wq
echo alias kvim='env NVIM_APPNAME=kickstart.nvim nvim' >> ~/.bashrc
$ source ~/.bash_profile
```

### Post Installation
Start kickstart.nvim
```sh
kvim
```
